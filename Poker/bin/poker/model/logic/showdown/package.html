This package contains all classes that are needed for the showdown. It contains a {@link poker.model.logic.showdown.HandWinner class} to determine the winner of a specific pot, which uses another {@link poker.model.logic.showdown.HandEvaluator class} which evaluates the hand for a player.

Additionally it contains a {@link poker.model.logic.showdown.HandQuality class} to evaluate card combinations and their winning chances. @author Becker
